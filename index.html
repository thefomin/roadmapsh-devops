Привет, GitHub Actions! 🚀

Этот сайт был автоматически развернут с помощью CI/CD пайплайна

📋 Информация о развертывании

Время развертывания:

Ветка: main

Триггер: Изменение index.html

Платформа: GitHub Pages + GitHub Actions

🔄 Как это работает

1. Изменения в файле index.html пушатся в ветку main

2. GitHub Actions автоматически запускает workflow

3. Сайт развертывается на GitHub Pages

4. Изменения становятся доступны по URL выше
