# 📊 Server Performance Statistics

Скрипт для мониторинга производительности Linux-сервера с цветным выводом в терминале.

## 📋 Описание

`server-stats.sh` - это bash-скрипт, который собирает и отображает ключевые метрики производительности сервера в реальном времени. Скрипт работает на любом Linux-сервере без необходимости установки дополнительного ПО.

## ✨ Возможности

- **CPU**: Текущая загрузка процессора
- **Память**: Общий объем, использованная и свободная память (в абсолютных значениях и процентах)
- **Диск**: Использование дискового пространства корневой файловой системы
- **Процессы**: 
  - Топ-5 процессов по загрузке CPU
  - Топ-5 процессов по использованию памяти
- **Дополнительно**:
  - Имя хоста
  - Время работы системы (uptime)
  - Количество активных пользователей
  - Средняя загрузка системы (load average)

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/thefomin/roadmapsh-devops.git
cd roadmapsh-devops
```

### 2. Сделать скрипт исполняемым
```bash
chmod +x server-stats.sh
```

### 3. Запустить скрипт
```bash
./server-stats.sh
```
или через bash:

```bash
bash server-stats.sh
```

## 🛠 Используемые команды

Скрипт использует стандартные утилиты Linux:

1. top - для получения информации о CPU и процессах

2. free - для статистики памяти

3. df - для информации о диске

4. ps - для детализации процессов

5. uptime - для времени работы и нагрузки

6. who - для подсчета пользователей
